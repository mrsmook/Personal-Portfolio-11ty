## Local Install

Clone, and Install with your _package-manager-of-choice_

```
$ yarn install
```

## Workflow

### First run

```
$ gulp clean # cleanup destination directory
$ gulp serve # generate, watch, and served to http://localhost:8080
```

### Next run

```
$ gulp serve
```
